======
 URLs
======
The application provides separate URLs for different use cases.
The basic :mod:`cloud_browser.urls` are used for installing the application
in the user-visible portion of a site, while :mod:`cloud_browser.urls_admin`
is appropriate for integrating the application into your admin.
